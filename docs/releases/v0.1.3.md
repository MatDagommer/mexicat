# Version v0.1.3

This release focuses on improving the security and reliability of score submissions by introducing stricter validation and enhanced error handling.

## New Features

- Added stricter validation for score submissions, including upper bounds for stage and consistency checks for elapsed time. Introduced MAX_STAGE and STAGE_DURATION constants to enforce these rules. (1a9238) (MatDagommer)
- Implemented detailed error responses and anomaly logging for invalid score submissions to aid in debugging and monitoring. (1a9238) (MatDagommer)
