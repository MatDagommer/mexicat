# Version v0.1.1

This release introduces improvements to the user interface layout and significantly enhances the JavaScript package release workflow. It also includes minor repository housekeeping.

## New Features

- Revamped the JavaScript package release workflow with improved versioning, dry-run support on pull requests, automated release notes generation, and more granular release steps. This makes the release process clearer, safer, and more automated. (aba9cb) (MatDagommer)

## Bug Fixes

- Adjusted the name entry overlay positioning in the CSS for a better layout, moving it from vertical centering to bottom alignment with a 4% offset. (15fa09) (MatDagommer)

## Deprecations

- Removed the .DS_Store file from the repository to prevent tracking of unnecessary macOS system files. (1e50f9) (MatDagommer)
