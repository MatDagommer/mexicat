# Version v0.1.2

This release introduces several major improvements to gameplay, user interface, and leaderboard security. Notable highlights include a new patch notes modal, enhanced leaderboard integrity with server-side anomaly detection, pause/resume functionality, and various visual and usability refinements.

## New Features

- Added a modal overlay to display patch notes to users, with dismiss logic and keyboard support. The modal prevents game input while visible and only appears if the user hasn't seen the latest notes within a set expiry window. (f53169) (MatDagommer)
- Implemented secure leaderboard score submission using a Supabase edge function, moving anomaly detection logic to the server for improved integrity. The frontend now submits additional metadata, and the release workflow includes Supabase CLI setup. (8a189c) (MatDagommer)
- Introduced pause and resume functionality for the game, allowing players to pause with Escape and resume with Enter or Space. A paused overlay is rendered when the game is paused. (eecb8b) (MatDagommer)
- Updated the background image asset to a new version for improved visuals. (ccfb86) (MatDagommer)
- Added display of player rank and total leaderboard entries after game over, including ordinal formatting and retrieval from Supabase. (b4c438) (MatDagommer)
- Implemented client-side anomaly detection to prevent impossible scores from being submitted to the leaderboard, based on elapsed play time and stage. (be784e) (MatDagommer)

## Bug Fixes

- Clarified patch note text regarding leaderboard rank and score box overlap, improved patch note box layout, and fixed dismissal logic to rely on expiry rather than localStorage. (333360) (MatDagommer)

## Deprecations

- Removed client-side computeMaxPossibleScore and related logic from leaderboard.js, as anomaly detection is now handled server-side. (8a189c) (MatDagommer)
- Removed the subtitle displaying creator and version from the title screen for a cleaner look. (6da4ab) (MatDagommer)
